CMDA Datasets

Data Structure: monthly, latitude-longitude-pressure gridded data

Temporal Resolution: monthly

Temporal Range: various (10-50 years)

Spatial Resolution: various (1, 0.5, 0.25 degrees)

Geographic Range: global

Dimensions: time, latitude, longitude, pressure

File Format: NetCDF

Types: model outputs, reanalysis datasets, observational datasets

  • Model Outputs:

    • CMIP5 historical runs (20+ models), CMIP5 AMIP runs, CMIP5 projection runs
    • CMIP6 historical runs (20+ models)
    • WRF runs

  • Reanalysis Datasets: ECMWF, GLDAS

  • Observational Datasets:

    • In-Situ Data: ARGO, World Ocean Database
    • Satellite Data: AIRS, SMAP, QuikSCAT, CERES, GRACE, TES, MODIS, MISR, CALIPSO, GPCP, MLS, CloudSat, AMSR-E, TRMM, GPM, ISCCP, AVISO

Variables: various ocean, atmosphere, and land variables

Number of Datasets: 2,145 sets

Total Data Volume: 2.15 TB

How to Search and Access: use the CMDA service and type keywords (e.g. AIRS) in the Search Filename/varNames/title section. It will show the list of datasets that meet the search criterion.
